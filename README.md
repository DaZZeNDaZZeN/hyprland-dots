#### keybindings:
<p>
Exit hyprland: SUPER + M <br> 
Close window: SUPER + Q <br>
Open terminal: SUPER + T or CTRL + ALT + T <br>
Open browser: SUPER + E <br>
Open app launcher: SUPER + R <br>
Next workspace with open windows: ALT + TAB <br>
Previous workspace with open windows: ALT + SHIFT + TAB <br>
Move window to next workspace: ALT + CTRL + TAB <br>
Move window to previous workspace: ALT + CTRL + SHIFT + TAB <br>
Open workspace {n}: SUPER + {n}  (SUPER + 1 for first, SUPER + 2 for second and so on) <br>
Move window to workspace {n}: SUPER + SHIFT + {n} <br>
Make window float: SUPER + B <br>
Screenshot: SUPER + SHIFT + S <br></p>


#### required packages for arch linux:

wm: hyprland
terminal: alacritty
file manager: dolphin
browser: firefox
app launcher: fuzzel
screenshot: hyprshot hyprpicker
auido: wireplumber
brightness: brightnessctl
multimedia: playerctl
fonts: noto-fonts

command with everything: sudo pacman -S hyprland alacritty dolphin fuzzel firefox wireplumber brightnessctl playerctl noto-fonts hyprshot hyprpicker
