i vibe coded most of quickshell config
hyprland config were written by myself

keybindings:
Exit hyprland: SUPER + M 
Close window: SUPER + Q
Open terminal: SUPER + T or CTRL + ALT + T 
Open browser: SUPER + E 
Open app launcher: SUPER + R
Next workspace with open windows: ALT + TAB
Previous workspace with open windows: ALT + SHIFT + TAB 
Move window to next workspace: ALT + CTRL + TAB
Move window to previous workspace: ALT + CTRL + SHIFT + TAB
Open workspace {n}: SUPER + {n}  (SUPER + 1 for first, SUPER + 2 for second and so on)
Move window to workspace {n}: SUPER + SHIFT + {n}
Make window float: SUPER + B 
Screenshot: SUPER + SHIFT + S 


required packages for arch linux:

wm: hyprland
terminal: alacritty
file manager: dolphin
browser: firefox
app launcher: fuzzel
screenshot: hyprshot hyprpicker
auido: wireplumber
brightness: brightnessctl
multimedia: playerctl
fonts: noto-fonts

command with everything: sudo pacman -S hyprland alacritty dolphin fuzzel firefox wireplumber brightnessctl playerctl noto-fonts hyprshot hyprpicker
